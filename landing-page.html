How it works

Configure, compile, deploy.

Read the docs →
01

Configure your sources

Point to any URL or local file. Mix adblock and hosts formats. Apply per-source transformations.

"sources": [
  {
    "source": "https://easylist.to/easylist/easylist.txt",
    "type": "adblock"
  }
]
02

Choose your transformations

Select from 11 built-in transformations — deduplicate, validate, compress, strip comments, and more.

"transformations": [
  "Deduplicate",
  "Validate",
  "Compress",
  "RemoveComments"
]
03

Deploy or call the API

Use the streaming API for real-time output, batch mode for up to 10 lists, or embed the library directly.

// Streaming compilation
POST /compile/stream

// Embed as library
import { compile }
  from "@jk-com/adblock-compiler"

Get started

Up and running
in minutes.

Install from JSR, configure your sources, and compile. Or deploy the full Worker and call it over HTTP. Either way, you're done before your VPN subscription renews.

View full docs → JSR package
example.ts
// Install: deno add @jk-com/adblock-compiler import { FilterCompiler, TransformationType } from "@jk-com/adblock-compiler"; const compiler = new FilterCompiler(); const result = await compiler.compile({ name: "My Privacy List", sources: [ { source: "https://easylist.to/easylist/easylist.txt", type: "adblock", }, { source: "https://someonewhocares.org/hosts/hosts", type: "hosts", }, ], transformations: [ TransformationType.Deduplicate, TransformationType.Validate, TransformationType.Compress, ], }); console.log(`Compiled ${result.rules.length} rules`);
